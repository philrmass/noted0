# Noted
by Phil Mass

Deployed at [https://philrmass.github.io/noted/](https://philrmass.github.io/noted/)

### Build Instrucitons
- Clone the repo
- `npm install` to install dependencies
- `npm run start` to start the dev server
- `npm run deploy` to deploy to gh pages
